An end-to-end Machine Learning project that predicts California median house prices using a tuned **RandomForest Regressor**, served via a **FastAPI REST endpoint** with live Swagger UI documentation.

---

## 🎯 Project Highlights

| Feature | Detail |
|---|---|
| Algorithm | RandomForestRegressor (tuned) |
| CV RMSE | **$48,713** (±$761 across 5 folds) |
| Train R² | **0.9749** |
| Tuning | RandomizedSearchCV — 20 iterations, 5-fold CV |
| API | FastAPI + Uvicorn + Pydantic validation |
| Dataset | California Census 1990 — 20,640 districts |

## ⚙️ Setup & Installation

```bash
# 1. Clone the repository
git clone https://https://github.com/AbuShadabkhan/California-housing-predictor
cd california-housing-predictor

# 2. Install dependencies
pip install -r requirements.txt

# 3. Train the model (runs once, saves model.pkl + pipeline.pkl)
python main.py

# 4. Run inference on saved test set
#    (delete model.pkl first to re-train, or just run to predict)
python main.py
```

## 🧠 ML Pipeline

```
housing.csv
    │
    ▼
Stratified Split (income_cat, 80/20)
    │
    ▼
ColumnTransformer
    ├── Numerical  → SimpleImputer(median) → StandardScaler
    └── Categorical → OneHotEncoder
    │
    ▼
RandomForestRegressor
    │
    ▼
RandomizedSearchCV (20 iter, 5-fold CV)
    │
    ▼
Best Model → model.pkl + pipeline.pkl
    │
    ▼
FastAPI /predict endpoint
```

## 📈 EDA Highlights

Key findings from `EDA_California_Housing.ipynb`:

- **`median_income`** is the strongest predictor with a Pearson correlation of **+0.69** with house value
- Only **1 column** has missing values — `total_bedrooms` (207 rows, ~1%) — handled by median imputation
- **`ISLAND`** districts have the highest median value (~$380k); **`INLAND`** the lowest (~$124k)
- Engineered feature **`bedrooms_per_room`** shows a strong negative correlation (−0.25) — more bedrooms relative to total rooms indicates lower-quality housing
- Expensive clusters are clearly visible around **SF Bay Area** and **Los Angeles** in geographic plots

- Target: `median_house_value` (range: $14,999 – $500,001)
